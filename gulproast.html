SARAH MAHL

GULP ROAST COFFEE

Gulp Roast needs a website for e-commerce to expand their business. With my website design, they’ll boost profits and improve brand loyalty.

OBJECTIVES

  • Design and wireframe the website.
  • Prepare the website for “developer handoff.”
  • Build the website, either with custom code or a no-code tool.

IMPORTANT LINKS

  • Live Website

FINAL WIREFRAMES

This project only required a home page and a 404 page, but with more time, I would make a complete website, perfect for Gulp Roast’s business needs.

DESIGN METHODS

To start, I needed inspiration, so I looked at other coffee companies to see how they structured their websites--a competitive analysis of sorts. I took my main inspiration from Death Wish Coffee Co. and Stumptown Coffee Roasters.

Death Wish Coffee Co.

Stumptown Coffee Roasters

After user testing, I was able to start coding. I tested no-code tools before deciding on custom code, and using Bootstrap, I was able to build the website quickly.

CONCLUSIONS

One of my main takeaways? Test, test, test. Both the design process and coding are extremely tedious, but by devoting my time, I was able to create a live, user-backed website for Gulp Roast.
